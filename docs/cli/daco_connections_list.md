## daco connections list

List all connections in the OpenDPI spec

### Synopsis

List all connections defined in the OpenDPI spec with their protocols and hosts.

```
daco connections list [flags]
```

### Examples

```
  # List connections
  daco connections list
```

### Options

```
  -h, --help   help for list
```
