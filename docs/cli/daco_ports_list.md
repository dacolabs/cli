## daco ports list

List all ports in the OpenDPI spec

### Synopsis

List all ports defined in the OpenDPI spec.
Displays port names, schema types, descriptions, and connection information.

```
daco ports list [flags]
```

### Examples

```
  # List ports
  daco ports list
```

### Options

```
  -h, --help   help for list
```
